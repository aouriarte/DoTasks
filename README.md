# DoTasks

Primer proyecto completo (front & back) con TypeScript

## 🗒️ Enunciado

La idea es crear una aplicación de tareas, poder crearlas, verlas, editarlas y eliminarlas.

## 🛠️ Tecnologías

- **TypeScript**
- **HTML**
- **Tailwind**
- **React.js**
- **Redux.js**
- **Node.js**
- **Express.js**
- **Sequelize.ts**
- **PostgreSQL**
